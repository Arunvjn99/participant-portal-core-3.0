Enhance the Auto Increase configuration screen to make the contribution growth easier to understand and more interactive.

The current design relies on button presets and a text-based timeline preview, which requires users to mentally simulate how their contribution will grow over time.

The goal is to introduce interactive controls and visual projections to improve clarity and engagement.

---

INCREASE CONTROL

Replace the current button-only options for yearly contribution increase with a slider control.

Label:

Increase contribution each year

Range:
0% to 3%

Display the selected value dynamically (e.g., 1% per year).

Retain quick preset buttons below the slider for common selections:

1% / year
2% / year
3% / year

---

MAX CONTRIBUTION CONTROL

Replace the button group for maximum contribution with a slider control.

Label:

Stop increasing when contributions reach

Range:
10% to 20%

Display the selected maximum value dynamically.

---

PROJECTION CHART

Replace the text-based timeline preview with a visual projection chart.

The chart should show contribution percentage growth over time.

X-axis:
Years

Y-axis:
Contribution percentage

The chart should visually show the progression from the current contribution (e.g., 8%) to the selected maximum (e.g., 15%).

The chart should update dynamically when either slider is adjusted.

---

PROGRESSION SUMMARY

Below the chart display a short summary:

Example:

Your contribution will increase from 8% to 15% over approximately 7 years.

---

FINANCIAL IMPACT

Add a simple comparison visualization showing projected retirement savings with and without automatic increases.

Example:

Without auto increases
$1.62M

With auto increases
$1.67M

Highlight the additional savings gained from enabling automatic increases.

---

MOBILE-FIRST LAYOUT

On mobile screens, stack elements vertically in this order:

Increase slider
Max contribution slider
Projection chart
Impact summary
Continue button

The chart should expand to full width for better readability.

---

UX GOAL

Users should be able to visually understand how their contribution will grow over time and easily adjust the settings using sliders while seeing real-time updates to the projected growth path and retirement impact.
