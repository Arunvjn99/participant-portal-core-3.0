UNIVERSAL TRANSACTION WIZARD

All transactions must follow a consistent step structure.

Step 1 – Transaction Intent
User selects transaction type and sees eligibility information.

Step 2 – Transaction Details
User enters transaction information such as amount or type.

Step 3 – Source Selection
User chooses contribution sources or investment funds.

Step 4 – Allocation or Amount
User specifies amounts or percentages.

Step 5 – Payment or Documents
User selects payment method or uploads required documents.

Step 6 – Review and Confirm
User reviews full summary before submission.
