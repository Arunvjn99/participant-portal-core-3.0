You are a senior fintech product designer designing an **Investment Portfolio page for a US Retirement Participant Portal (401k platform)**.

The design must follow a **clear UX narrative** and work for both **new investors and experienced users**.

The page should answer these user questions in order:

1. How much money do I currently have?
2. How is my portfolio performing?
3. How is my money allocated?
4. What does this mean for my retirement?
5. What actions should I take?

Use a **mobile-first design** that scales to tablet and desktop using a responsive grid.

Avoid placing retirement projections at the top. The user must first understand their portfolio.

---

PAGE STRUCTURE

SECTION 1 — PORTFOLIO SNAPSHOT (Hero)

Purpose: Immediately show the user's financial position.

Display the following metrics:

• Total Portfolio Balance
• Total Invested Amount
• Total Gain / Loss
• Annual Return %

Also show a small comparison indicator:

• vs S&P 500 benchmark

Use a large numeric display for the portfolio balance.

This should be the **primary visual focus of the page**.

---

SECTION 2 — PORTFOLIO PERFORMANCE

Purpose: Help users understand how their money is growing.

Include:

• Portfolio Growth Line Chart
• Benchmark comparison line (S&P 500)

Time filters:

• 1M
• 3M
• YTD
• 1Y
• 5Y

The chart should be simple and readable.

---

SECTION 3 — PORTFOLIO ALLOCATION

Purpose: Show how investments are distributed.

Display:

• Donut chart of allocation

Group funds into understandable categories:

• Growth Assets (Stocks)
• Income Assets (Bonds)
• Defensive Assets (Cash / Stable)

Include percentage labels and category explanations.

---

SECTION 4 — RETIREMENT PLANNING

Purpose: Show the impact of current investments on retirement.

Display:

• Projected retirement savings at age 65
• Estimated monthly retirement income
• Retirement readiness score

Include scenario buttons:

• Base scenario
• Market -10%
• Market +10%
• Increase contribution

The projection chart should show:

• current trajectory
• retirement goal line

---

SECTION 5 — RECOMMENDED ACTIONS

Purpose: Guide the participant toward better retirement outcomes.

Show 3 action cards:

1. Rebalance Portfolio
   Description: Align with target allocation.

2. Increase Contribution
   Description: Boost retirement savings.

3. Review Risk Level
   Description: Adjust investment risk profile.

Each card must include:

• short explanation
• expected improvement
• clear CTA button

---

DESIGN RULES

Follow these UX principles:

• Clear information hierarchy
• Progressive disclosure
• Low cognitive load
• Data first, insights second, actions last

Use modern fintech design patterns similar to:

• Vanguard
• Fidelity
• Wealthfront

Visual style:

• clean neutral background
• card-based layout
• clear typography hierarchy
• accessible color contrast

Ensure the page works well on:

• mobile screens
• tablet
• desktop dashboards
