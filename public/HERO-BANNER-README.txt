To use your Whisk hero image on the test dashboard:

1. Open: https://labs.google/fx/tools/whisk/share/5e90633qs0000
2. Download or save the image (e.g. right-click → Save image as, or use the tool's export/share option).
3. Save it in this folder as: hero-whisk.png

The pre-enrollment test page (/test/pre-enrollment-dashboard) will use hero-whisk.png as the hero banner. If the file is missing, it falls back to the default image.
